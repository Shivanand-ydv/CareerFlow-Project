# 🚀 CareerFlow – Job Application Tracker

> **A responsive Job Application Tracker built with HTML, CSS, and JavaScript to manage, organize, and track job applications with dashboard analytics and Local Storage.**

CareerFlow is a responsive web application that helps users organize their job search by managing applications, tracking their progress, and analyzing application statistics. The project features a clean and modern interface with real-time dashboard updates, search and filtering functionality, and browser-based data persistence using Local Storage.

---

## 📌 Features

- ✅ Add new job applications
- ✅ Track application status
- ✅ Dashboard with live statistics
- ✅ Search applications
- ✅ Filter applications by status
- ✅ Delete applications
- ✅ Analytics section
- ✅ Local Storage support
- ✅ Responsive Design
---

## 🚀 Functionalities

### Add Application

Users can add job application details including:

- Company Name
- Job Role
- Location
- Salary
- Job Link
- Apply Date
- Status
- Notes

---

### Dashboard

Displays real-time statistics:

- Total Applications
- Applied
- Interview
- Offer
- Rejected

---

### Search & Filter

Users can:

- Search by Company Name
- Search by Job Role
- Filter applications by Status

---

### Analytics

Provides useful insights including:

- Total Applications
- Interview Count
- Offer Count
- Success Rate

---

### Local Storage

All application data is stored in the browser using **Local Storage**, allowing data to persist even after refreshing the page.
